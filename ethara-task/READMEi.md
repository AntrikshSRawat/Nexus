# Ethera Task - Full Stack Task Management App

## 🚀 Overview
A full-stack task management application where users can create projects, assign tasks, and track progress with role-based access control.

## 🔥 Features
- User Authentication (Signup/Login)
- Role-based Access (Admin / Member)
- Project Management
- Task Assignment & Tracking
- Dashboard with task insights

## 🛠 Tech Stack
Frontend: React (Vite), Tailwind CSS  
Backend: Node.js, Express  
Database: MongoDB Atlas  

Deployment:
- Frontend: Vercel
- Backend: Railway

## 🌐 Live Demo
https://ethera-task.vercel.app

## ⚙️ Setup Instructions
1. Clone repo
2. Install dependencies
3. Add environment variables:
   - MONGODB_URI
   - JWT_SECRET
4. Run backend and frontend
